# 🎹 Open Survivor (오픈 서바이버)

사용자가 직접 게임의 외형을 커스터마이징하고, 끊임없이 몰려오는 적들로부터 살아남는 **웹 기반 로그라이크 서바이벌 게임**입니다.

## 🚀 주요 특징 (Key Features)

*   **실시간 스킨 커스터마이징:** 게임 중 언제든지 플레이어와 적의 이미지를 업로드하여 바꿀 수 있습니다. 업로드된 이미지는 자동으로 깔끔한 **원형 크롭** 처리가 되어 게임 규격에 맞춰 적용됩니다.
*   **다양한 무기 시스템:**
    *   **자동 소총:** 가장 가까운 적을 조준하여 자동으로 사격합니다. (기본 무기)
    *   **위성 검:** 플레이어 주변을 공전하며 다가오는 적들을 베어 넘깁니다. (최대 6개)
    *   **라이트닝:** 무작위 위치의 적에게 강력한 낙뢰를 떨어뜨립니다.
*   **로그라이크 성장 시스템:** 레벨업 시 3가지 무작위 강화 카드 중 하나를 선택하여 나만의 빌드를 구성할 수 있습니다.
*   **단계별 보스 레이드:**
    *   **미니 보스:** 매 5, 15, 25분 등 5분 단위로 거대한 중간 보스가 출현합니다.
    *   **스테이지 보스:** 매 10, 20, 30분 등 10분 단위로 강력한 스테이지 보스가 소환됩니다.
*   **자석 & 등급 보석 시스템:** 
    *   경험치 보석 근처에 가면 자석처럼 보석이 플레이어에게 빨려 들어옵니다. (초기 범위 60px)
    *   보석 등급(Cyan, Green, Purple, Gold)에 따라 획득하는 경험치 양이 다릅니다.
*   **지능형 난이도 조절:** 시간이 경과함에 따라 적들의 스폰 속도와 체력, 이동 속도가 점진적으로 상승하며, 특정 시간대별로 밸런스가 정밀하게 조정되어 있습니다.

## 🎮 조작 방법 (How to Play)

*   **이동:** `W`, `A`, `S`, `D` 또는 **방향키**
*   **공격:** 자동 조준 및 자동 발사 (무기 획득 시)
*   **강화:** 경험치 바를 채워 레벨업 시 나타나는 선택지 중 하나를 클릭
*   **일시정지:** 우측 하단 `||` 버튼 클릭으로 게임 중지 및 재개
*   **관리자 콘솔:** 우측 상단 레벨(Lv) 텍스트 클릭 시 테스트용 메뉴 오픈

## 🛠️ 기술 스택 (Tech Stack)

*   **Engine:** Phaser 3 (JavaScript Game Engine)
*   **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
*   **Deployment:** Vercel
*   **Version Control:** GitHub
